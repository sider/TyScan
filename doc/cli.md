# Command line options

## Scan your TypeScript code

```
Usage: scan [options] [path...]

Options:
  -c, --config <path>  path to configration file (default: "tyscan.yml")
  -j, --json           output json
```

## Test your patterns

```
Usage: test [options]

Options:
  -c, --config <path>  path to configration file (default: "tyscan.yml")
```
